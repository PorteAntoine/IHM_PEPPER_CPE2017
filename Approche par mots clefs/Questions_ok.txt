Questions auxquelles on sait déja répondre :
	- localisation d'un objet dans l'arène (where are ... situated?)
	- couleur d'un objet (what is the color of ...?)
	- catégorie d'un objet (to which category belong ...?)
	- description d'une pièce (which objetcs are stored in ...?)
	- description d'un objet (type fridge, desk ...) (which objetcs are stored in ...?)
	- localisation d'une catégorie (where are ... situated?)
	- demander à manger
	- demander à boire
	- comparaison entre deux objets :
		* (do ... and ... belong to the same category?)
		* (between ... and ..., which one is the ... (smaller, bigger, lighter, heavier)?)
		* the biggest object ?
	- quantification d'objets :
		* (how many objects are stored in ...?)
		* (what can i find in the ...?)
	- combien d'objet de (categorie) (how many snack ...)
	- combien de hommes/femmes (how many ... crowd ?)
	- description des personnes (how are ... crowd ?)
	- combien de personnes debouts/assis/couchées (how many ... standing/lying/sitting ?)
	- description des personnes debouts/assis/couchées
	- demander où se situe un (loc_env) (Where is the located the ... ? In which room is the ... ?)
	- demander si un (loc_env) est un placement (Is ... a placement ?)
	- demander si un (loc_env) est un beacon (Is ... a beacon ?)
